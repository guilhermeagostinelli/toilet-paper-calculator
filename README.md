# Toilet Paper Price Calculator

A web-based application designed to help consumers determine the most cost-effective toilet paper. It normalizes prices based on regional metrics—such as sheets per roll in the USA or meters per roll in Brazil—so you can always find the best deal.

## Features

- **Regional Normalization:** Accurately compares prices between different standardizations (e.g., sheets per roll vs. meters per roll).
- **Modern UI:** Features a sleek, dark-themed, glassmorphism-inspired design.
- **Internationalization (i18n):** Supports English and Portuguese (PT-BR) with automatic language detection based on the user's browser.
- **Dynamic Pricing Input:** Includes a fast, masked input field that handles currency formatting and dynamic comma/dot parsing.

## Technologies Used

- Vanilla HTML5
- CSS3 (Vanilla CSS with modern features, variables, and glassmorphism styling)
- JavaScript (Vanilla JS for logic, localization, and dynamic formatting)

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in your favorite modern web browser.
3. No build tools or backend servers are required to run the basic application.

## How It Works

Simply input the price, number of rolls, and the length/sheet count according to the input fields. The application will instantly calculate the normalized value, allowing you to easily compare different brands and packaging sizes.
